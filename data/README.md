# Company Profile Content

Edit `company-profile.json` to update the public website without touching page HTML.

Practical workflow:

1. Upload replacement images to `assets/img/custom/`.
2. Edit paths, titles, paragraphs, service cards, testimonials, contact email, and WhatsApp number in `company-profile.json`.
3. Keep paths relative, for example `assets/img/custom/foto-workshop.webp`.
4. Reload `https://dsitujayabersama.com/`.

The static HTML remains as fallback if the JSON file fails to load.
