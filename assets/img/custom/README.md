# Custom Media

Drop replacement company profile images in this folder, then update `data/company-profile.json`.

Recommended sizes:

- Hero image: 1920x1080 WebP or JPG.
- Service card image: 800x520 WebP or JPG.
- Logo/OpenGraph image: 1200x630 WebP or PNG.

Example:

```json
"backgroundImage": "assets/img/custom/hero-company-yard.webp"
```
